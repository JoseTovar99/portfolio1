    Orbitir

    Orbitir is a space traveling app prototype. The objective was to create an app that resembles a normal flight traveling layout but with more futuristic features. The main page consists of a popular planets section with details under about availability, the rocket used, and how long the journey will take.

    At the bottom of the page is another option that lets the user schedule an orbit of the Earth. The main color is a beige to signify a more luxurious interface.
